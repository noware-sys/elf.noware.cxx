A C++ library; still in development.


Notable components:

noware::container::any (not a true any! can only container a string or a number at any time)
noware::container::object (a recursive, associative and dynamically-allocated array; like a JSON object)
noware::container::db::sql::sqlite (a sqlite3 wrapper)
noware::net::cluster (a manager of a list of network nodes)
noware::computer (an AI)
noware::machine (a VM)
